Ramses Magazine Website
=======================

Files:
- index.html        Homepage
- admin.html        Front-end admin dashboard mockup
- styles.css        Main styles
- app.js            Homepage logic + Arabic/English switch + saved customizations
- admin.js          Dashboard logic with localStorage

Important:
- This version is front-end only.
- Admin panel saves changes in browser localStorage.
- Later, it can be connected to a real backend like Supabase or Firebase.

How to upload to GitHub Pages:
1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repository.
3. In repository settings, enable GitHub Pages from the main branch.
4. Your site will open from the generated GitHub Pages link.
